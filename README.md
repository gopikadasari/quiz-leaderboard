# Quiz Leaderboard System

SRM Internship Assignment - Quiz Leaderboard System

## What this does
- Polls the quiz API 10 times
- Removes duplicate events using roundId + participant as unique key
- Calculates total score per participant
- Submits sorted leaderboard

## How to run
1. Make sure Java 17 is installed
2. Compile: javac QuizLeaderboard.java
3. Run: java QuizLeaderboard
